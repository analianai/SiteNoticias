# 📰 Site de Notícias do CRP

Este repositório contém o desenvolvimento do **site de notícias do
CEEP Áureo FIlho**, uma plataforma digital criada para organizar, publicar e divulgar
conteúdos informativos produzidos pela equipe editorial.

O objetivo do projeto é estruturar um ambiente moderno e acessível onde
seja possível **centralizar notícias, reportagens, vídeos, imagens e
outros conteúdos jornalísticos**, permitindo que a comunidade acompanhe
atualizações de forma rápida e organizada.

O site foi desenvolvido utilizando **HTML, CSS e Bootstrap**, garantindo
uma interface **responsiva, moderna e de fácil manutenção**, permitindo
que futuras atualizações sejam realizadas com agilidade.

O projeto tem como finalidade:

# 🏗 Estrutura do Site

O site foi organizado em seções principais que seguem a lógica de
portais de notícias e agências jornalísticas.

Entre os principais componentes estão:

### Página inicial

Apresenta as notícias mais recentes, destaques e conteúdos principais do
portal.

### Seção de notícias

Área dedicada à publicação de matérias, reportagens e conteúdos
informativos.

### Conteúdos multimídia

Espaço destinado à publicação de imagens, vídeos e materiais
complementares.

### Área de navegação

Menu superior estruturado para facilitar o acesso às diferentes seções
do site.

### Rodapé institucional

Contém informações institucionais, direitos autorais e links adicionais.

# 💻 Área de Desenvolvimento

Este repositório também contém a **área de desenvolvimento do projeto**,
onde estão sendo estruturadas as funcionalidades e melhorias do site.

A área de desenvolvimento é responsável por:

-   Estruturar o **layout inicial do portal**
-   Organizar os **componentes visuais**
-   Definir a **arquitetura das páginas**
-   Testar funcionalidades antes da publicação
-   Implementar melhorias contínuas

Essa área funciona como um **ambiente de construção e testes**, onde
novas funcionalidades podem ser desenvolvidas e avaliadas antes de serem
integradas ao site oficial.
